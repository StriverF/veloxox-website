Documentation d'aide

Comment utiliser Velo Dash ?

1. Ajouter une vidéo : Cliquez sur "Ajouter une vidéo" pour importer une vidéo depuis la galerie.

2. Importer des données de cyclisme : Cliquez sur n'importe quel onglet en bas pour importer des données de cyclisme (si des données sont déjà importées, cliquez pour ajouter les composants correspondants au tableau de bord de la vidéo).

3. Choisir des données de cyclisme : Sélectionnez pour importer des données depuis un fichier fit ou Apple Health (la synchronisation avec les comptes tiers est en développement).

4. Ajouter des composants : Choisissez les données du tableau de bord que vous souhaitez ajouter à la vidéo depuis les onglets inférieurs.

5. Agencement des composants : Déplacez les composants sur la vidéo en les faisant glisser ou en utilisant un geste de pincement pour les agrandir/réduire (les gestes peuvent être effectués dans toute la zone vidéo, pas seulement sur le composant sélectionné).

6. Alignement temporel : Vérifiez si l'alignement est automatique dans le premier onglet. Sinon, utilisez l'outil d'alignement de la ligne temporelle pour ajuster manuellement les données de cyclisme avec le temps de la vidéo (ajoutez un composant temporel ou un itinéraire à la vidéo, puis ajustez l'outil d'alignement pour trouver le moment correspondant).

7. Exporter la vidéo : Cliquez sur le bouton d'exportation en haut à droite pour sauvegarder la vidéo dans la galerie.

Comment obtenir un fichier .fit avec des données de cyclisme ?

Compteur Garmin : Connectez-vous à Garmin Connect, cliquez sur les trois lignes en haut à gauche > Activités > Toutes les activités > Sélectionnez l'enregistrement à exporter > Cliquez sur l'engrenage > Exporter les données brutes > Téléchargez et enregistrez sur votre téléphone.

Compteur Magene : Ouvrez l'application "Magene Utility", trouvez les données de cyclisme correspondantes, puis cliquez sur le bouton de partage en haut à droite sur la page des détails. Choisissez "Partager le fichier fit" dans les options de partage et enregistrez-le à l'emplacement souhaité.

Autres compteurs : Contactez le support officiel de votre compteur pour connaître la méthode d'exportation des données de cyclisme.

Partager un fichier fit pour ouvrir Velo Dash : Vous pouvez ouvrir des fichiers `.fit` dans Velo Dash en sélectionnant "Ouvrir avec une autre application" dans une conversation WeChat, puis en choisissant Velo Dash. Dans des applications de compteurs vélo comme OneLap Sports ou iGPSPORT, vous pouvez également partager des fichiers `.fit` directement vers Velo Dash pour importer les données correspondantes. De plus, vous pouvez ouvrir des fichiers `.fit` directement depuis l'application Fichiers pour lancer Velo Dash et importer les données.

Comment sauvegarder une disposition personnalisée du tableau de bord ?

Une fois satisfait de votre disposition, cliquez sur l'onglet "Configuration" > Sélectionnez "Modèles de mise en page" > Cliquez sur l'icône "+" > Entrez le nom du modèle > Confirmez.

Lors des futures importations, vous pourrez appliquer vos modèles enregistrés d'un seul clic.

Difficulté d'alignement temporel ?

Les vidéos prises avec des smartphones ou des caméras sportives peuvent être automatiquement alignées avec la ligne temporelle. Pour assurer cet alignement :

1. Ne découpez pas les vidéos exportées depuis des caméras sportives, car cela peut désynchroniser les temps d'enregistrement.

2. Lors de l'envoi ou du transfert des vidéos, choisissez l'option "Vidéo originale", car la compression peut effacer les métadonnées temporelles.

3. Évitez de renommer les fichiers vidéo. En cas d'absence des métadonnées, le nom du fichier peut être utilisé pour deviner le temps d'enregistrement.

4. Vérifiez les métadonnées depuis la galerie système. Si les temps ont été modifiés, restaurez-les aux valeurs originales.

Si l'alignement automatique échoue, vous pouvez aligner manuellement avec ces astuces :

1. Si vous connaissez précisément l'heure de capture, ajoutez un composant temporel au tableau de bord, puis ajustez l'outil d'alignement jusqu'à la correspondance.

2. Ajoutez des composants de vitesse, cadence ou puissance, puis mettez la vidéo en pause lorsque les pieds s'arrêtent ou quand vous entendez le cliquetis des roues. Ajustez ensuite l'outil d'alignement jusqu'à ce que la cadence ou la puissance atteigne 0.

3. Repérez des virages dans la vidéo. Mettez la vidéo en pause à ces endroits, puis ajustez l'outil jusqu'à ce que l'itinéraire montre le virage correspondant.

4. Repérez des montées ou descentes dans la vidéo. Mettez la vidéo en pause, puis ajustez l'outil pour faire correspondre les changements de pente.
